# Leprosy_powerBI-Dashboard
Welcome to the Leprosy Data of India Dashboard repository! This project presents an interactive Power BI dashboard designed to analyze and visualize leprosy statistics across India. The dashboard transforms raw healthcare data into meaningful insights, enabling users to explore state-wise disease distribution, prevalence rates, yearly trends, and treatment outcomes through dynamic and interactive visualizations. It aims to support healthcare professionals, researchers, and policymakers in understanding disease patterns and making informed public health decisions.

The Leprosy Data of India Dashboard provides a comprehensive and interactive analysis of leprosy statistics through intuitive visualizations. It presents key healthcare indicators such as Total New Cases Detected and Total Cases Discharged as Cured using KPI cards, while an interactive bubble map displays the geographical distribution of cases across different states and union territories to identify high-burden regions. The dashboard also includes state-wise prevalence analysis, year-wise trend analysis (2017–18 to 2021–22), and case category distribution through charts that highlight new cases, cases under record, discharged cases, and deleted records. Dynamic State/UT filters and responsive visualizations allow users to perform detailed regional analysis, making data exploration simple, interactive, and insightful.

Technologies Used
1. Power BI Desktop: The primary platform used to design and develop the interactive dashboard, enabling advanced data visualization, report creation, and insightful healthcare analytics.
2. Power Query: Utilized for data cleaning, transformation, and preprocessing to ensure data consistency, accuracy, and readiness for analysis.
3. DAX (Data Analysis Expressions): Used to create calculated measures, KPIs, and custom metrics that provide deeper analytical insights and enhance dashboard functionality.
4. Data Modeling: Implemented to establish relationships between multiple datasets, optimize data performance, and ensure efficient querying within Power BI.
5. Microsoft Excel: Used as the primary data source for storing, organizing, and preparing the healthcare dataset before importing it into Power BI for visualization and analysis.
6. Power BI Maps & Interactive Visuals: Leveraged built-in map visuals, KPI cards, donut charts, scatter plots, slicers, and other interactive visualizations to provide an engaging and user-friendly analytical experience.

The primary objective of this project is to demonstrate the effective use of Business Intelligence (BI) in healthcare analytics by transforming raw leprosy data into meaningful visual insights. The dashboard enables healthcare professionals, researchers, and policymakers to monitor disease prevalence, analyze regional trends, evaluate treatment outcomes, identify high-risk areas, and support evidence-based decision-making for improved leprosy surveillance and public health planning across India.
